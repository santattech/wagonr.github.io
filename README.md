## Quick Start

1. Clone the repository
2. Open `index.html` for GPS tracking
3. Open `wagonR.html` for dashboard analytics
4. Allow location permissions for full functionality

## File Structure

- `index.html` - Main GPS tracker interface
- `wagonR.html` - Vehicle dashboard and analytics
- `tracker.js` - GPS tracking logic
- `mileageChart.js` - Chart rendering for analytics
- `manifest.json` - PWA configuration
- `service-worker.js` - Offline functionality
